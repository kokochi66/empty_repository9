# 젠킨스 학습 스터디


## 젠킨스란?

- 소프트 웨어 개발에 필요한 통합 서비스를 제공하는 툴 (CI 툴)


## 젠킨스의 장점

- 빌드 자동화
- 컴파일 오류 검출
- 테스트 자동화 (버전관리 시스템과 연동)
- 정적 코드 분석으로 코딩 규약 준수 여부 체크
- 성능 변화 감시
- 배포
- 배치 작업 간략화
